# cuticulus

This repository is for the project "Ant Cuticle Texture Analysis".

## Installation

### Requirements

This project utilizes [poetry](https://python-poetry.org/) for building and running.

```bash
git clone 'https://github.com/ngngardner/cuticulus'
cd cuticulus
poetry shell
poetry install
```
